## Introducing Today's Project

### Key Tools and Concepts
<p>The key tools I used include multi-factor authentication and verification. Key concepts I learnt include how to properly set up an account and enable my account to use AWS services.</p>

## Creating My AWS Account
<p>In this step, I'm going to sign up for a new AWS, verify email address, and create a strong root user password.</p>

### Understanding the root user
<p>I learned that a root user is the account I'll be using for AWS services. A strong password is important because it ensures security.</p>

## Setting Up Billing and Payment
<p>In this step, I'm going to choose the free account plan, enter my contact information, and add a payment method so I can verify I'm a real person and be accountable for charges in my account.</p>

### Why AWS requires a payment method
<p>I learned that AWS requires a payment method because it gives the user a time frame for using AWS services before it closes. The user can experiment using free credits, learn, and navigate the website within the six months time frame.</p>

<br>

## Completing Account Activation
<p>In this step, I'm going to confirm my identity and select a support plan </p>

### Identity verification and support plan
<p>I verified my identity by using my BDO debit card. The Basic support plan includes access to a personal health dashboard and a trusted advisor, and self-service access to AWS resources.</p>
